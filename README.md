# Simple PyKV
Simple, lightweight Python-based key-value storage server that allows you to save and retrieve project-specific data over a RESTful HTTP API.

Designed to be easy to setup, use and maintain, Simple PyKV is perfect for small to medium-sized projects that require a straightforward way to manage key-value pairs without the overhead of a full-fledged database system.

## Features
- **Lightweight**: Minimal dependencies and easy to deploy.
- **RESTful API**: Simple HTTP endpoints for storing and retrieving data.
- **Project-specific storage**: Isolate key-value pairs by project for better organisation.
- **In-memory storage with optional persistence**: Fast access with the option to save data to disk.
- **Easy to use**: Simple API design for quick integration into your projects.
